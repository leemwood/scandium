# Scandium

> [!WARNING]
> **This mod is entirely AI-driven.** Before using this mod, you should know that all of its code and logic were written by AI. If you are interested, feel free to fork it for your own development branch. Additionally, if you wish to submit a PR, please be aware that I will continue to update the project using AI, and your changes may be modified or even overwritten.

Scandium is a high-performance rendering optimization mod for Minecraft Fabric 1.21.11, designed to work seamlessly with Sodium. It implements advanced culling techniques to improve frame rates, especially in complex environments.

## Features

- **Advanced Culling**: Includes FOV-based culling and aggressive mountain culling to reduce the number of rendered chunks.
- **Sodium Integration**: Configuration is directly integrated into Sodium's Video Settings screen. No extra mod menus required.
- **Performance Focused**: Minimal overhead with maximum impact on rendering efficiency.

## Installation

1. Ensure you have [Fabric Loader](https://fabricmc.net/) installed.
2. Download and install [Sodium](https://modrinth.com/mod/sodium).
3. Place the Scandium jar file in your `.minecraft/mods` folder.

## Configuration

Scandium's settings can be found in **Options > Video Settings > Scandium**.
